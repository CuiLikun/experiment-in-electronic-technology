# experiment-in-electronic-technology
USTC EE2002.01 2024Fall   中国科学技术大学 EE2002.01 2024 秋季：本仓库收录 10 份电子技术核心实验文件，涵盖三大类实验： 电路特性实验：含一阶 RC 电路（零输入 / 零状态响应、积分 / 微分 / 脉冲分压）、RC 电路频率特性（低通、串并联选频）、晶体管共射极放大电路（静态工作点、增益、失真分析）； 器件与应用实验：包括集成运放应用（比例、加减、积分等电路）、门电路测试（与 / 或 / 非等门功能验证）、编码器译码器（数码管驱动、功能扩展）； 综合系统设计：含抢答器设计（抢答模块、分频、倒计时）及配套思考题（仪器操作、电路原理、误差分析）。 每份文件均含实验步骤、数据处理、误差分析与思考题，可直接用于实验参考、报告撰写。
